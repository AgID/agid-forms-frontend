# Comunicazione esiti dei test di usabilità

Puoi utilizzare questo form, per inviare ad AGID i risultati dei test di usabilità, come previsto dal Piano Triennale ICT 2020-2022:

- OB.1.2 - Migliorare l’esperienza d’uso e l’accessibilità dei servizi
- R.A.1.2b - Diffusione dei test di usabilità previsti dalle Linee Guida AGID per il design dei servizi nelle amministrazioni per agevolare il feedback e le valutazioni da parte degli utenti.

Per maggiori dettagli consulta il <a class="external-link" href="https://docs.italia.it/italia/piano-triennale-ict/pianotriennale-ict-doc/it/2020-2022/capitolo-1-servizi/cosa-devono-fare-le-pa.html" target="_blank">Piano Triennale</a>

Se non hai ancora svolto i test o se comunque desideri capire come fare i test di usabilità e presentare i risultati,
qui puoi trovare tutte le informazioni e indicazioni pratiche:

- <a class="external-link" href="https://designers.italia.it/kit/usability-test/">https://designers.italia.it/kit/usability-test/</a>
- <a class="external-link" href="http://www.funzionepubblica.gov.it/glu">http://www.funzionepubblica.gov.it/glu</a>
