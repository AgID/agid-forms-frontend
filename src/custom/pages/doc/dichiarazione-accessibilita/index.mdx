# Dichiarazione di accessibilità

I soggetti erogatori devono compilare e pubblicare, esclusivamente tramite
questa piattaforma, la  dichiarazione di accessibilità per ciascun sito web e
applicazione mobile di cui sono titolari.

- [Compila nuova dichiarazione](/form/dichiarazione-accessibilita)
- [Approfondimenti e riferimenti normativi](https://www.agid.gov.it/it/design-servizi/accessibilita-siti-web)
